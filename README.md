<h1 align="center">Weather - Learning React.js</h1>

## Getting started 

    # Clone repository
    $ git clone https://github.com/matheustanaka/react

    # Install dependencies
    $ yarn install

    # Open the project
    $ cd weather-react

    # Start project
    $ yarn start

    # You can see the project on localhost link
    $ http://localhost:3000/
